Material Design Button 4/26/18

Angular Material's buttons look fantastic. You can see how clicking a button causes a bounce-like effect, while a circular ripple originates from where the user clicked.

Drawing on Angular Material's source code as inspiration, I aimed to replicate the behavior of the buttons, using a minimal implementation that still demonstrates how the buttons actually work.

How It Works

The key idea behind achieving the ripple effect is that, when a user clicks the button a "ripple" element is added to the DOM. The element receives a "ripple" class, which provides the following main styles:

.ripple border-radius 50% position absolute transform scale(0) top 0px left 0px transition all .4s cubic-bezier(.25,.8,.25,1) z-index -1 background hsla(196, 89%, 50%, 0.41)

The ripple is initially given a scale of zero, rendering it invisible. A style recalculation of the element is forced, and then its scale is set to 1. Over a period of .4s the ripple transitions from scale(0) to scale(1) (its full size), resulting in the growing ripple effect. When the transitionend event fires (meaning the element has grown to its full size) it is then removed from the DOM.
