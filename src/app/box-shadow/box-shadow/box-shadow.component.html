BOX SHADOW 9/12/17

The objective of this demonstration was to show how CSS' box-shadow property could be used in conjunction with animations to achieve a nice "hover" affect when a user hovers over a button.

.slick-btn { border: none; border-radius: 3px; color: white; font-size: 22px; font-weight: 300; padding: 0.25em 0; position: relative; text-align: center; text-shadow: 0 0 2px rgba(0, 0, 0, 0.5); transition: all 0.2s ease-in-out; width: 200px; } .slick-btn:hover { box-shadow: 2px 4px 8px 0 rgba(0, 0, 0, 0.3); }
