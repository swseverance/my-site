TEXT SHADOW 9/2/17

The text shadow property accepts four arguments:

1. Horizontal offset of the shadow (positive or negative number) 2. Vertical offset of the shadow (positive or negative number) 3. Blur radius (The larger the value the blurrier the shadow becomes) 4. Color (hexadecimal, rgb, rgba, etc.)

Multiple text shadows can be applied to the same element by using comma separated values. Check out this fun example:

.miami { color: white; font-size: 96px; text-shadow: 3px 0 4px orange, 0 3px 4px magenta, -3px 0 4px lime, 0 -3px 4px magenta; }
