Creating a function to invert a binary tree is a great whiteboarding exercise. This model demonstrates how a binary tree can be inverted by creating a binary tree with arbitrary values, inverting the tree, and highlighting each node after its left and right children are swapped.

The code to recursively invert a binary tree is surprisingly simple and elegant.

const invertBinaryTree = (node: Node): Node => { if (node) { const invertedLeftSubTree = invertBinaryTree(node.left); const invertedRightSubTree = invertBinaryTree(node.right); node.left = invertedRightSubTree; node.right = invertedLeftSubTree; } return node; }
